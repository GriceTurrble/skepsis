Remote usability testing was conducted from July 1-10th, 2020. 9 participants, ages 50-75 were chosen due to their closeness in age bracket and computer literacy of current MyCommunitySite.com members and target clients. Participants were asked to go to MyCommunitySite.com, join the site, and navigate through the signup process.

  • Unfamiliar UI and lack of information led users to become confused and ask for guidance.
  • There are several dead ends and error pages with no way to know what the issues are or recover.
  • The sign up process took up to 12 minutes to complete for some users.
  • Many users were confused by the payment page, unsure of what each plan is.
  • Users did not understand what the page and theme selection was due to lack of information.
  • Many users were confused by the payment page, unsure of what each plan is.
  • All users felt the process was very straightforward.
  • No major changes are needed to improve the sign up process - mostly simplification.

Homepage

  • 7 of 9 users did not have issues finding the “Join Now” link at the top of the page.
  • 2 users scrolled up and down on the page looking for a way to join.
  • One user spent almost 2 minutes looking for a way to join the site.

Recommendations:

Make the Join Me link more prominent in the top right corner by styling it as a button. Larger font, bright colors, and a background. Change the name from "Join me" to "Sign Up"

Sign Up

  • 7 of 9 users put an email address or “www.” “.com” as their site name
  • Disclaimer and TOS links are broken
  • One user was concerned about the security of the website and their password

Recommendations:

  • Show instructions for site name and password upfront with more clear, understandable language
  • Use more common UI standards
  • Fix error for Disclaimer and Terms of Service
  • Redesign error response, allow user to go back or see error without having to use browser back bar
  • Make error messages more prominent and explanatory
  • Add more introductory text - possibly merge with next screen

Getting Started

Users did not spend much time reading this page, and did not have trouble moving forward.

Recommendations:

Consider removing this page or minimizing the amount of text.

Select Theme

  • Users didn't understand the differences between the choices.
  • One user had issues selecting any other theme than the first with their browser (IE10)
  • Error links and links in the agreements section are broken.

Recommendations:

  • Make larger and more detailed images of the themes available.
  • Test the entire site for cross-browser compatibility.
  • Create pages for the TOS and agreements

Association Details

No issues or recommendations for this page.

Select Pages

Some users were not sure what these pages meant, or what they would be for. Others could not select additional pages at all.

Recommendations:

  • Include additional information about pages, or remove this step until after the user joins.
  • Address bugs by making sure the website is cross-browser compatible.

Add Members

Users get an error when clicking save and next if they have not uploaded a file. There is no way to recover from this error page. Also, users were unsure what member lists were, or what they had to upload.

Recommendations:

  • Make errors throw modal or pop up windows that users can recover from. Include the reason for the error in the window.
  • Remove this step until after the user signs up for the service.

Pick a Plan

Users were generally unsure what the plans were and what they were paying for. One user thought they needed to type in the plan they wanted.

Recommendations:

  • Add more information about the available plans and their cost.
  • Use more common UI patterns for inputs, drop downs, and buttons.
